# 💸 Control de Gastos - React Native

Una aplicación móvil multiplataforma diseñada para el control financiero personal. Permite a los usuarios registrar ingresos y egresos de manera intuitiva y eficiente.

## 🚀 Tecnologías
* **Framework:** React Native (Expo/CLI)
* **Lenguaje:** TypeScript (97.6%)
* **Estado:** Context API / Hooks
* **Estilos:** Flexbox & Native Styles

## 🛠️ Estructura del Proyecto
* `/components`: Componentes reutilizables de la interfaz.
* `/context`: Manejo del estado global de la aplicación.
* `/services`: Lógica de conexión con APIs o persistencia.
* `/hooks`: Lógica personalizada para el manejo de gastos.

## 📲 Instalación
1. `npm install`
2. `npx react-native run-android` o `run-ios`
